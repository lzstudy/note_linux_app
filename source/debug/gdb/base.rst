1 GDB调试方案介绍
-------------------

GDB调试方案有4种, 推荐使用 ``GDB+GDBSERVER+VSCODE`` 和 ``GDB``

========================= =========== ================================
方法                      调试级别    描述
========================= =========== ================================
GDB                       符号级      移植简单, 只需要移植GDB就能调试
GDB + GDBSERVER + [UI]    源码级      调试简单, 需要移植GDBSERVER
========================= =========== ================================

.. note:: 
    
    常用UI工具vscode、vim、peda、gef
